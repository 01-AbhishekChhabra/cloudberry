# Cloudberry Matrix 

[Cloudberry](http://cloudberry.ics.uci.edu) is now using the [Play! Framework](https://www.playframework.com/)

## Build

Prerequisites: scala, sbt

### To compile projects
```
 sbt compile
```

### Run projects
```
 sbt ";project neo; run "
```

You can replace the `neo` to other sub-project to run specific project.
